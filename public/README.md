# Public Assets Folder

This folder contains static assets that are served as-is by Vite. Files in this folder are accessible at the root URL.

## Required SEO Assets

### Favicons
- `favicon.ico` - Standard favicon (16x16, 32x32, 48x48)
- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon
- `apple-touch-icon.png` - 180x180 Apple touch icon

### Social Media Preview Images
- `og-image.png` - Open Graph image (1200x630px recommended)
- `twitter-image.png` - Twitter card image (1200x600px recommended)

### PWA Icons
- `icon-192.png` - 192x192 PWA icon
- `icon-512.png` - 512x512 PWA icon

### SEO Files
- `robots.txt` - Search engine crawling instructions
- `sitemap.xml` - Site structure for search engines
- `manifest.json` - PWA manifest file

## Generating Icons

You can use tools like:
- [Favicon Generator](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [PWA Asset Generator](https://www.pwabuilder.com/)

## Notes
- All images should be optimized for web
- Use PNG format for transparency support
- Ensure proper dimensions for each platform
